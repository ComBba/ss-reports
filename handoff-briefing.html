01 · 어떤 서비스인가

Social Seeding v2는 브랜드가 던진 한 줄 브리프를 받아, TikTok 인플루언서 캠페인의 전체 운영 루프를 에이전트 플릿이 대신 굴리는 제품입니다. 대시보드는 수작업 화면이 아니라 Mission Control — 에이전트가 한 일의 타임라인 + 승인 인박스 — 이 됩니다.

brand brief ─▶ coordinator (gemini-3.5-flash, Vertex global) └▶ 22-agent ADK fleet : 소싱 → 검증 → 아웃리치 → 응답처리 → 발행 → 콘텐츠검증 → 리포트 └── A2A v0.3 message:send ──▶ OSS tiktok-mcp-server (크리에이터 소싱 · 브랜드 자산) ▲ 사람은 켜둔 정책 게이트에서만 개입 (기본 always_ask)
항목내용
형태멀티테넌트 B2B SaaS 인플루언서-캠페인 플랫폼 (per-call MCP 티어 과금)
핵심 가치에이전시 관리 수수료 대비 약 99.7% 저렴, TAM $1.15B (BUSINESS-CASE.md)
에이전트22개 전문 에이전트 — 자유 ReAct 루프가 아니라 큐레이션된 도구 + Zod/typia 출력 계약 + USD 캡 + 에스컬레이션을 가진 함수
모델Gemini 3.5 flash(판단·코디네이터) / 3.1 flash-lite(벌크) — Vertex global 엔드포인트 only (D53)
안전장치모든 외부 발신(gmail.send)은 정책 게이트 통과 전 미발화; 사용자 입력은 prompt-guard 선통과
